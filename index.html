BLAEO Generators v1.21

A tool for generating bars, game boxes, panels and progress bars for use with BLAEO posts.

Steam forum post - post your feedback, suggestions and bug reports there

Developed by kubikill
for Backlog Assassins Extraordinaire

Thanks to mandrill for the help with the autofilling feature!

Changelog:


v1.21 - March 29, 2019

  • Bar and Panel: Added screenshot button
  • Added "Copy with linebreaks to clipboard" buttons to export easier to read code
  • Added hex color codes to "Completion bar color"

v1.2 - May 25, 2018

  • Added a panel generator
  • Added Markdown support to reviews
  • Added automatic and manual saving for reviews
  • Bar: Added "click button to view review"
  • Progress bar: Added striped/animated bars
  • Miscellaneous small changes

v1.1 - April 21, 2018

  • Added a progress bar generator
  • Bar: Added reviews
  • Bar: Added option to change game image position
  • Bar: Added option to change completion bar position
  • Bar: Text color can now be set separately for title and playtime/achievements
  • Bar: Text color option now affects achievement link
  • Changed how games with no achievements are handled
  • Fixed achievement text showing "undefined of x achievements", if user has 0 achievements in a game
  • Fixed fetching games with progress set to "never played" or "won't play"
  • Implemented some tiny changes I don't remember anymore
  • Removed the "preview always visible" option because it breaks review previewing ¯\_(ツ)_/¯

v1.01 - March 27, 2018

  • Added an one-time welcoming message
  • Added "never played (white)" to completion bar colors
  • Fixed achievement links pointing to "undefined" profiles, and not updating properly
  • Fixed AppIDs not updating properly

v1.0 - March 18, 2018

  • Initial release
